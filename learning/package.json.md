# npm package.json

## package.json : Specifics of npm's package.json handling

每个项目的根目录下面，一般都有一个package.json文件，
定义了这个项目所需要的各种模块，以及项目的配置信息（比如名称、版本、许可证等元数据）。
npm install命令根据这个配置文件，自动下载所需的模块，也就是配置项目所需的运行和开发环境。
